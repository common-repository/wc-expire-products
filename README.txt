== Description ==

Defines a expiration time to products and send an email notifying the author.

With this plugin you can:

*   Define a time (in days) for products expire
*   Notify 5 days before the product will expire
*   Notify on the same day in which the product expires
*   Notify 7 days after the product expired
*	Remove the product 14 days after expired
*	Also remove images (only featured image and gallery)

== Installation ==

1. Upload `wc-expired-products` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter the plugin options to configure.
